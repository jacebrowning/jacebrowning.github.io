# Overview

Website for [jacebrowning.info](https://jacebrowning.info/).

## Requirements

* Ruby 2.4.9

## Usage

Run the site:

```
$ make run
```

or additionally open in a browser:

```
$ make launch
```

Validate the content:

```
$ make check
```
